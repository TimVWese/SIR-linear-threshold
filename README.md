## Dependencies

Dependencies are in `Project.toml`. To install them, run

```julia
using Pkg
Pkg.activate(".")
Pkg.instantiate()
```

`General` and `SIRLT` are not part of the Julia general registry. To install them, run

```julia
using Pkg
Pkg.add(url="https://github.com/TimVWese/General.jl")
Pkg.add(url="")
```
